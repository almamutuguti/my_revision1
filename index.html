In Git..

Like I said before, it was a bit complicated. At the end, i got to understand;

  • How to make folders [mkdir]
  • How to print working directory [pwd]
  • How to change directory [cd]
  • How to make files [echo >] or [touch]
  • How to delete folders and files
    • Folders
      1. [rmdir] -folder-
      2. [rm -rf] -folder- to force delete
    • Files
      1. [rm] -file-
      2. [rm -rf] -file
  • How to list contents in a folder [ls]
  • How to create a repository
  • How to connect VsCode with your working folder [code .]
Warning! the delete tag should not be used aimlessly because important data can be erased and barely retraced.

GitHub
